# John_ML_Portfolio  
**Project Type:** Machine Learning & AI Portfolio  
**Tools Used:** Python, Scikit-learn, TensorFlow, Keras, NLP, Pandas, Matplotlib  
**Purpose:** To demonstrate hands-on machine learning skills across multiple domains  



#  Portfolio Overview

This portfolio contains a collection of machine learning projects completed across different chapters. Each project demonstrates key concepts such as data preprocessing, model training, evaluation, visualization, and deep learning. The goal is to show practical understanding of machine learning workflows.


# Projects Included

## 1. Machine Learning Review (Chapter 1)
Introduced core ML concepts including supervised vs unsupervised learning and basic workflows.

## 2. Classification (Chapter 3)
Implemented classification models such as Logistic Regression and KNN with performance evaluation metrics.

## 3. Support Vector Machines (Chapter 5)
Explored SVM models, kernel functions, and hyperparameter tuning for classification tasks.

## 4. Decision Trees (Chapter 6)
Built and visualized decision trees, analyzed overfitting, and explored feature importance.

## 5. Dimensionality Reduction (Chapter 8)
Applied PCA and t-SNE to reduce data dimensions and improve visualization of complex datasets.

## 6. Artificial Neural Networks (Chapter 10)
Developed deep learning models using Keras, including dense neural networks for classification tasks.

## 7. Natural Language Processing (Chapter 16)
Performed sentiment analysis using IMDB dataset with TF-IDF, Logistic Regression, and neural networks with embeddings.



#  Favorite Project

My favorite project is **Natural Language Processing (Chapter 16)** because it demonstrates how raw text can be transformed into meaningful numerical data and used for sentiment classification. It also introduced advanced concepts like embeddings and neural networks.

#  Reflection

Throughout this portfolio, I gained a strong understanding of how machine learning systems are built from raw data to final predictions. I learned how important preprocessing is for improving model performance and how different algorithms behave on different types of data.

Classification models like Logistic Regression and SVM performed well on structured data, while neural networks and NLP techniques were powerful for complex tasks like text analysis. Decision Trees provided interpretability, while ensemble methods improved accuracy.

Overall, this experience prepared me for real-world machine learning tasks by teaching me how to clean data, select models, evaluate performance, and interpret results effectively.
